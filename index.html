Welcome to the genomicsel project !

The genomicsel R package will consists of different functionalities that can be used for different methods of genomic predictions and genome-wide association mapping as part of single package along with available testing datasets.

What is machine learning

Machine learning is construction of algorithm or use of scientific learning from data. Such method include building model based on the inputs (known as training dataset) - not explicitly programmed instructions and use the models to make predictions or decision.

What is genomic or genome-wide prediction ?

Genomic prediction is application of machine learning tools that use the genomics data in making prediction or making decision.

What is genome-wide / genomics selection

Practicing plant or animal selection decisions based on genomics prediction.

What is genome-wide association (GWAS) analysis ?

Using genome-wide markers, GWAS is approach of finding genetics responsible for qualitative and quantitative traits. Such genes can be used for making selection decisions. Such approach usually use broad genetic material rather using a single family .

What is linkage based QTL mapping ?

Mapping of quantitative trait loci(QTL) based on linkage has same object that of GWAS, however this use a single or few families as population for genetic discovery.

Project objective:

The main objective of the project is to create R package platform that can perform different genomics analysis in single platform.
